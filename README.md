##  Project Description
This project is web portfolio that demonstrates different programming paradigms implemented directly in the browser.  

It includes interactive applications developed using HTML, CSS, and JavaScript.

---

##  Programming Paradigms

###  Imperative
**Movie Mood Recommender**  
Uses conditional logic if/else to recommend movies based on what mood and genre the user selects.

---

###  Functional
**Movie Search & Filter**  
Uses functional programming methods like:
- filter() → to select movies by genre or rating  
- map() → to transform movie data  
- random selection → to display a random movie  

---

###  Logical
**Movie Genre Recommender**  
Applies rule-based logic to return movie suggestions based on selected genre

---

###  Object-Oriented
**Movie Watchlist**  
Implements classes and objects:
- Movie class  
- MovieList class  

Allows users to:
- add movies  
- mark as watched  
- delete movies  

---

##  Extra Programs

###  Movie Runtime Planner
Dynamically generates input fields based on the number of movies and calculates total watch time.

---

###  Movie Rating App
Allows users to:
- enter a movie name  
- assign a star rating  
- display rated movies using star emoji  

---

##  Curriculum Vitae (CV)
The CV is dynamically generated from structured data and displayed in HTML.

Features:
- multiple categories (education, skills, experience, etc.)
- dynamic rendering
- PDF export functionality
